
# health_system_assignment
Створи Unity проект для реалізації системи здоров’я (HealthSystem) з використанням івентів 
для керування інтерфейсом користувача та звуковими ефектами.

Проект має містити:

+ Клас HealthSystem:
- [x] Поля: MaxHP і HP.
- [x] Метод TakeDamage, що зменшує значення HP.
- [x] Необхідно передбачити реакцію інших елементів гри на зміну HP.

+ UI Компоненти:

- [x] Слайдер, що відображає поточний стан HP у співвідношенні з MaxHP.
- [x] Кнопка "TakeDamage", яка зменшує HP при натисканні.

+ Івенти:

- [x] Оновлення слайдера повинно відбуватись автоматично при зміні HP.
- [x] Звук має відтворюватись тільки при зменшенні HP.
- [x] Івенти повинні бути ключовим механізмом для взаємодії між HealthSystem та UI/звуком. Заборонено 
- [x] напряму оновлювати UI з HealthSystem.

Проект необхідно завантажити на GitHub і надіслати посилання для перевірки.
