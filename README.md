
# health_system_assignment
Створи Unity проект для реалізації системи здоров’я (HealthSystem) з використанням івентів 
для керування інтерфейсом користувача та звуковими ефектами.

Проект має містити:

+ Клас HealthSystem:
- [ ] Поля: MaxHP і HP.
- [ ] Метод TakeDamage, що зменшує значення HP.
- [ ] Необхідно передбачити реакцію інших елементів гри на зміну HP.

+ UI Компоненти:

- [ ] Слайдер, що відображає поточний стан HP у співвідношенні з MaxHP.
- [ ] Кнопка "TakeDamage", яка зменшує HP при натисканні.

+ Івенти:

- [ ] Оновлення слайдера повинно відбуватись автоматично при зміні HP.
- [ ] Звук має відтворюватись тільки при зменшенні HP.
- [ ] 
Івенти повинні бути ключовим механізмом для взаємодії між HealthSystem та UI/звуком. 
Заборонено напряму оновлювати UI з HealthSystem.

Проект необхідно завантажити на GitHub і надіслати посилання для перевірки.
